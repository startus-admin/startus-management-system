# STARTUS アイデア・課題メモ

## 大会参加募集の現状ワークフロー

現在、陸上大会の参加募集は以下の手順で行っている：

1. 開催者の大会要項を確認
2. クラブ用PDF要項に書き起こし
3. スグラムアプリの新規募集ページで案内文章・内容・アンケートを作成
4. URLを発行し、配信メッセージで会員に配信（配信文章も作成）
5. 配信だけでは周知しきれないため、A4半分サイズの告知ビラ（デザイン性あり）を作成し、陸上会員に配布

---

## スグラムアプリの問題点

### 1. 新規募集ページの作成が手間
- HTMLコピペなどはできず、決まった入力画面・管理画面を毎回手動で設定しなければならない

### 2. URL配信が必要
- 同じアプリ内の機能なのに、わざわざURLを発行して配信メッセージで送らないといけない

### 3. 対象会員に常時表示されない
- 大会募集やイベントのタブで対象会員に常時表示させればいいのに、配信しないと見えない

### 4. 申込時に認証キーの入力が必要
- クレカ等を登録済みのアカウントでも、申込時に認証キーを入力しなければならない
- アプリ内から申し込めば、アカウント情報で自動入力されるべき

### 5. 申込情報がアプリ内で確認できない
- 申込データはメールで送信されるだけで、アプリ内で常時閲覧・管理できない

### 6. 大会参加履歴・記録がアプリで見れない
- クラブ側で大会の参加履歴や記録を別途管理しているが、アプリ内で会員ごとに確認できない
